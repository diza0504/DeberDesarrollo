EJERCICIOS SUPLEMENTARIOS - ANGULAR

HTML es excelente para declarar documentos estáticos, pero falla cuando intentamos usarlo para declarar vistas dinámicas en aplicaciones web. AngularJS le permite ampliar el vocabulario HTML para su aplicación. El entorno resultante es extraordinariamente expresivo, legible y rápido de desarrollar.

Otros marcos se ocupan de las deficiencias de HTML ya sea abstrayendo HTML, CSS y / o JavaScript o proporcionando una forma imperativa de manipular el DOM. Ninguno de estos aborda el problema de raíz de que HTML no fue diseñado para vistas dinámicas.

AngularJS es un conjunto de herramientas para crear el marco más adecuado para el desarrollo de su aplicación. Es completamente extensible y funciona bien con otras bibliotecas. Cada función se puede modificar o reemplazar para adaptarse a su flujo de trabajo de desarrollo único y a sus necesidades de funciones. Siga leyendo para descubrir cómo.
